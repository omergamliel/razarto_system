# סקירה כללית – RAZARTO

## מהו הפרויקט?
RAZARTO הוא יישום ווב לניהול משמרות והחלפות בתוך ארגון. המערכת מאפשרת לעובדים לצפות בלוח משמרות, להגיש בקשות להחלפה מלאה או חלקית, לאשר כיסויים, ולנהל תהליכי הרשאות. הממשק מותאם לעבודה מול משתמשים בארגון עם תפקידים והרשאות שונות.

## קהל יעד
- מנהלי משמרות ומנהלים תפעוליים.
- עובדים המבקשים להחליף/לכסות משמרות.
- צוותי תמיכה/תפעול הזקוקים לתמונת מצב (KPI, היסטוריה, בקשות פתוחות).

## טכנולוגיות עיקריות
- **React + Vite** להרצת אפליקציה מהירה בצד הלקוח.
- **React Router** לניהול ניווט בין דפים.
- **@tanstack/react-query** לניהול תצוגת נתונים, Cache ושכבות Fetch.
- **Base44 SDK** לתקשורת עם ה-Backend (Auth, Entities, Integrations).
- **Tailwind CSS** לעיצוב עם Utility Classes.
- **Radix UI + shadcn/ui** כבסיס לקומפוננטות UI.
- **framer-motion** לאנימציות ומיקרו-אינטראקציות.

## נקודות מפתח בהבנת המערכת
- כל גישה לנתונים מתבצעת דרך **Base44 SDK** (ראו `src/api/*`).
- אימות משתמשים וניהול סטייט גלובלי מתבצע דרך **AuthContext**.
- לוח המשמרות מנוהל בקומפוננטה המרכזית `ShiftCalendar`.
- פריסת הנתיב הראשי מוגדרת דרך `pages.config.js`.
